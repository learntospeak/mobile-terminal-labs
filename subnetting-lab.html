Quick Rules

  • Network = first address in the subnet
  • Broadcast = last address in the subnet
  • First usable = network + 1
  • Last usable = broadcast - 1
  • Usable range = everything in between

Commands

  • show run = current active config
  • show start = saved startup config
  • show version = IOS version and system details
  • show interface = interface details, errors, bandwidth
  • show flash = files stored in flash memory

Switch Mgmt

  • A normal switch IP is for remote management only.
  • A normal switch is not the default gateway.
  • Switch management IP is usually on a virtual interface.

Security

  • SSH = secure and encrypted
  • Telnet = insecure and clear text
