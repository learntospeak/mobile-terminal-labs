Gameplay: Tap connected nodes to guide the packet worm through the network. Correct path reaches FILESERVER and unlocks the Golden Egg.
1
Normal lab firstThe terminal lab works like usual. No obvious Easter egg button.
2
Curious discoveryA strange file or weird output hints at a secret command.
3
Mini-game opensA separate network map appears. Tap nodes, not quiz cards.
4
Golden Egg rewardReach the correct target, collect an egg, return to the lab.
Design rule: keep this standalone until it is stable. Later, connect it through a hidden file or subtle command only after standalone testing passes.
